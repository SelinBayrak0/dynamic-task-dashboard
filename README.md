# Dynamic Task Dashboard

This project is a dynamic task management dashboard built using vanilla JavaScript.

## File Organization

- index.html – Main HTML structure
- styles.css – Basic styling
- taskManager.js – Task and TaskManager classes
- api.js – Simulated async API
- main.js – DOM manipulation and event handling

## Challenges Faced

- Understanding immutable state updates
- Debugging script loading and file path issues
- Managing DOM re-rendering after state changes
